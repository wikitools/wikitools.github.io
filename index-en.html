About project

The aim of this project is to create an application for viewing Wikipedia's pages in a new and innovative way: by visualizing connections between articles. Additional tools - such as the timeline and the currently selected subpage preview - will allow for analysis of the presented data.

The project is conducted at the Gdańsk University of Technology Immersive 3D Visualization Lab as an bachelor thesis project.
